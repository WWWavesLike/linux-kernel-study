1.실습 내용:
    hello.ko 모듈을 컴파일하여 커널에 동적로드함.
2.발생 오류: struct module 크기(레이아웃) 차이로 인해 모듈 적재 거부.
    실제 커널 모듈 컴파일에 쓴 커널(디폴트 옵션 사용)과 qemu로 구동 중인 리눅스 커널(디버그 옵션들이 작동)이
    달라 발생한 오류다. 
    커널은 모듈 안에 들어있는 struct module의 크기를 커널 내부의 struct module과 비교하는데,
    모듈과 커널 내부 구조체 크기 차이가 발생하여 모듈 적재를 거부하였다.
    커널 이미지를 동일하게 맞춰 오류를 해결하였다.
3.결과 :
    hello.ko 모듈을 로드하고, 언로드하였다.
