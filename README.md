# linux-kernel-study

Linux kernel 6.17.9 기반에서 커널의 핵심 동작 방식을 이해하기 위해 진행한 실습 내용을 정리한 저장소다.

커널 모듈의 구조와 빌드 환경, 사용자 공간과의 인터페이스 구성 방식 등을 직접 구현하며 개발 역량 강화, 커널 이해 향상을 목표로 한다.

세부 분석, 실습 과정은 별도의 디렉토리에 정리되어있다.

---
## 프로젝트 개요

- Linux 커널 드라이버 개발의 기본 구조 및 워크플로우 이해
- 모듈 빌드, 로드 과정과 시스템 연동 방식 확인
- 커널 내부 동작을 직접 구현하며 기반 역량 확보

---

## 개발 환경

- Kernel 버전 : Linux 6.17.9
- OS : Debian(호스트), 순정 커널 + BusyBox(가상환경)
- Toolchain : GCC, Makefile, KVM+QEMU

---

## 디렉토리 구조
- README.md : 저장소 설명
- .clangd : 커널 라이브러리 설정
- .clang-format : 커널 코딩스타일 포맷팅 설정
- 0.template : 실습 진행을 위한 템플릿
- 1.hello-module : 기초적인 모듈 로드, 언로드 실습
- 2.interrupt : 인터럽트 발생, 전반기 처리, 후반기 처리 실습
