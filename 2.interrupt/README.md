1.실습 내용:

    인터럽트를 발생시켜 전반부 처리(top half)와 후반부 처리(bottom half)를 작동시키는 실습을 하였다.
    인터럽트는 char device 파일을 만들어 해당 파일에 write()을 사용함으로써 발생시켰다.
    후반부 처리는 태스크릿(tasklet)을 사용하여 구현하였다.
<br>
2.결과 :

    intrpt.ko 모듈을 컴파일하여 가상환경에 적재시킨 후, char device 파일에 echo을 쏘아
    인터럽트를 발생시켰다. 인터럽트에 의해 전반부 처리가 진행되어 로그가 발생되고,
    전반부 처리에서 태스크릿으로 수행할 작업(함수)를 예약한다. 
    이후 예약된 후반부 작업이 진행되어 로그가 출력되고 종료된다.
